角色详情接口，客户端点击新增角色或角色权限设置（roleInfo1.0）
-----------------------------
>请求参数:

    {
        roleId:角色id（非必传，为空代表新增，只返回菜单列表，否则代表修改，返回菜单列表与当前角色信息）
    }
